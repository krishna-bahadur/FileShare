# FileShare (.NET Core MVC Project)

This is a simple ASP.NET Core MVC project for file sharing.

## Features
- ASP.NET Core MVC
- .NET 8
- Shared File Upto 250 MB
- Auto Delete Files after 12 Hours of Uplaod 
- Secured link
